Eliminación de datos de usuario

Responsable: Poliedro Software SAS
Aplicación: Poliedro Zammad Facebook / PQRS Digital

Poliedro Software SAS respeta la privacidad de los usuarios y permite solicitar la eliminación de los datos personales asociados al uso de nuestras aplicaciones, integraciones y servicios digitales.

¿Cómo solicitar la eliminación de datos?

Para solicitar la eliminación de datos personales asociados a nuestra aplicación, el usuario puede enviar una solicitud al siguiente correo:

poliedrosoftware@gmail.com

La solicitud debe incluir:

  • Nombre completo
  • Correo electrónico asociado
  • Número de teléfono, si aplica
  • Nombre de la aplicación o servicio relacionado
  • Solicitud expresa de eliminación de datos

Tiempo de atención

Una vez recibida la solicitud, Poliedro Software SAS revisará y procesará la eliminación de los datos en un plazo razonable, conforme a la normatividad aplicable de protección de datos personales.
